02 KK_SETUPCONTRIBUTION — TWO-ROUND PRF PROTOCOL

Click a trustee to compromise it.

Round 1: trustee t computes Rt = PRF_R(seed_t, keyId), CHKt = PRF_Chk(seed_t, keyId). Aggregator: CRV.R = XOR(R_base, R₀, R₁, …). A coalition of k−1 trustees observes a set of PRF outputs that are jointly uniform — the masked key is indistinguishable from random. With fewer than k active trustees, KK_Aggregator_Sign() aborts.

05 _XOR_MANY() — MEASURED ON YOUR MACHINE

Byte loop vs Uint32Array. Measured live.

Python's int.from_bytes(v,'big') ^ acc is a single C-level operation — the interpreter dispatches once regardless of length. A byte loop dispatches 16 times per 16-byte value. JS equivalent: viewing Uint8Array as Uint32Array reduces 16 XOR iterations to 4 (one 32-bit word per iteration). Click below to measure the actual speedup on this machine.

# int XOR: one C-level op regardless of byte count
def _xor_many(values):
    if not values: return b""
    n = len(values[0])
    acc = int.from_bytes(values[0], "big")
    for v in values[1:]:
        if len(v) != n:
            raise ValueError("length mismatch")
        acc ^= int.from_bytes(v, "big")
    return acc.to_bytes(n, "big")

hash.py· hash_lms() — domain-separated SHA-256
def hash_lms(tweak, *data):
    h = hashlib.sha256()
    for t in tweak:          # each int → 4-byte big-endian prefix
        if isinstance(t, int):
            h.update(t.to_bytes(4, "big"))
        else:
            h.update(t)
    for d in data:
        h.update(d)
    return h.digest()

def KK_SetupContribution(seed, KeyID, R, path_lens, sk_shape):
    rows, cols = sk_shape
    return {
        "Rt":    PRF_R(seed, KeyID, N_BYTES),       # tweak (30, keyId)
        "CHKt":  PRF_Chk(seed, KeyID, N_BYTES),     # tweak (31, keyId)
        "Auth":  PRF_Auth(seed, KeyID, R),          # tweak (32, keyId)
        "PATHt": [PRF_Path(seed, KeyID, idx, n)
                  for idx, n in enumerate(path_lens)],
        "SKt":   [[PRF_Chain(seed, KeyID, i, j, N_BYTES)
                   for j in range(cols)]
                  for i in range(rows)],
    }
